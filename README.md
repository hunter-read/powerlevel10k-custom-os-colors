# powerlevel10k-custom-os-colors
A custom color override for powerlevel10k os icons

Install simply with
```
sh -c "$(curl -fsSL curl -fsSL https://raw.github.com/hunter-read/powerlevel10k-custom-os-colors/master/install.sh)"
```
